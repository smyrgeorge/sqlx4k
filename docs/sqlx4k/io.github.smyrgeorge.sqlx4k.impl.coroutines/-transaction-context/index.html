TransactionContext

class TransactionContext(tx: Transaction) : CoroutineContext.Element, Transaction(source)

Represents a coroutine-aware transactional context that integrates a transaction as part of the coroutine context.

This class provides a mechanism to manage transactional operations within a coroutine, allowing the use of transaction functions seamlessly while maintaining coroutine context safety. It implements CoroutineContext.Element to operate as part of the coroutine context and delegates all transaction operations to the provided Transaction instance.
