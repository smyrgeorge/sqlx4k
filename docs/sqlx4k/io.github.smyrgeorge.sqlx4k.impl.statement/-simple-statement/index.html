SimpleStatement

open class SimpleStatement(sql: String) : Statement

Represents a simple SQL statement that supports binding values to both positional and named parameters. This class provides functionality to render the final SQL string with all parameters replaced by their bound values.

Parameters

sql

The SQL statement string that may contain positional ("?") and named (e.g., ":name") parameters to be bound.
