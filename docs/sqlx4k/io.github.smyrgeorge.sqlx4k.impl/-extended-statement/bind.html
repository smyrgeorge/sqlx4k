open override fun bind(index: Int, value: Any?): ExtendedStatement

Binds a value to a positional parameter in the prepared statement.

Return

The current PgStatement instance with the bound parameter, allowing method chaining.

Parameters

index

The zero-based positional index of the parameter to bind.

value

The value to bind to the specified parameter.

Throws

DbError

if the provided index is out of bounds of the statement's parameters.
