Column

data class Column(val ordinal: Int, val name: String, val type: String, value: String?)(source)

Represents a column in a database table.

This class provides metadata and access to the value of a column in a database row. It includes properties to retrieve information such as the column's ordinal position, name, type, and value, along with methods to manipulate or retrieve the column's value.
