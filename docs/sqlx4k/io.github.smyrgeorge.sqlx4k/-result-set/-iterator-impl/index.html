class IteratorImpl(rs: ResultSet) : Iterator<ResultSet.Row> (source)

Implementation of the Iterator interface for iterating over rows in a ResultSet.

This class provides mechanisms to traverse the rows in a result set, ensuring that any errors in the result set are detected and propagated appropriately.
