ConnectionPool

interface ConnectionPool(source)

Interface representing a connection pool for managing and reusing database connections.

A ConnectionPool is responsible for managing the lifecycle of a set of database connections, enabling efficient reuse and minimizing the overhead of establishing new connections. It provides methods for acquiring and releasing connections, managing pool size, and configuring the pool behavior.
