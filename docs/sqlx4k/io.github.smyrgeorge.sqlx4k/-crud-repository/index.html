interface CrudRepository<T>(source)

Interface defining a contract for basic CRUD (Create, Read, Update, Delete) operations on a data source.

This interface abstracts common operations to be implemented for handling entities of type T. Each operation is asynchronous and returns a Result, encapsulating either the successful result or an error in case of failure.

T

The type of the entity managed by the repository.
