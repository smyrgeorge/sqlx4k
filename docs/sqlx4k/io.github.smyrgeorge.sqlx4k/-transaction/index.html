Transaction

interface Transaction : QueryExecutor(source)

Represents a transaction in the system, providing methods to manage and execute transactional operations such as commit and rollback.

This interface integrates with the Driver interface to facilitate execution of SQL queries and retrieval of results within a transactional context.
