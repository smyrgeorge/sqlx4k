Executes an SQLx operation asynchronously using Kotlin coroutines.

This function transforms a callback-based SQLx operation into a suspending function, allowing for more readable asynchronous code. It creates a stable reference to the continuation and passes its pointer to the callback function, which is expected to resume the continuation when the operation completes.

Return

The SQLx result pointer, or null if the operation did not produce a result

A function that takes a pointer to the continuation and initiates the SQLx operation. This function should eventually resume the continuation with the result pointer.
