class Sqlx4kSqldelightDriver<T>(driver: T)

A driver implementation for SQLDelight using an underlying driver that supports connection pooling and transactions.

Implementation based on: https://github.com/cashapp/sqldelight/blob/master/drivers/r2dbc-driver/src/main/kotlin/app/cash/sqldelight/driver/r2dbc/R2dbcDriver.kt
